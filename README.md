### 环境准备

node v16.14.2
pnpm 8.0.0

### 项目初始化创建

```
pnpm create vite
```

### eslint配置

```
pnpm i eslint -D
```

### 生成 eslint配置文件

```
npx eslint --init
```

### 生成 .husky

```
npx husky --init
```

### eslint fix

```
pnpm run fix
```

### .prettierrce配置

```
pnpm run lint
```

### 代码格式化

```
pnpm run format
```

### 快速创建vue3组件模版

v3ts + 回车

## 使用仓库相关命令

### 安装 pnpm

```
npm i pnpm -g
```

### 安装所有依赖

```
pnpm install
```

### 启动本地测试项目

```
pnpm run dev
```

### 打包

```
pnpm run build
```
